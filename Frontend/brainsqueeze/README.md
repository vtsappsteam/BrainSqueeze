# Brain Squeeze

This template should help get you started developing with Vue 3 in Vite.

# Directory Structure Description

* public/: This directory contains public files of the application.

* src/: This directory contains the source code of the application. 
    * assets/: Images, icons, and other resources used throughout the application.
    * components/: Vue components used throughout the application and layouts for pages
    * router/: Router configuration.
    * services/:
        * Users/: Users management.
        * config/: Axios configuration.
    * styles/: Stylesheets used throughout the application. 
        * base/: Base styles, variables, overrides.
    * views/: Main Vue components representing pages in the application. 
    * App.vue: Main Vue component serving as the root template of the application.
    * main.js: Main file that initializes the Vue application.

* .gitignore: File containing a list of files and directories to be ignored by git.

* node_modules/: Directory where npm packages installed for this project reside.

* package.json: File containing the project configuration and a list of npm packages installed.
